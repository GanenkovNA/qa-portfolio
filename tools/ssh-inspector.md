## SSH Inspector — инструмент для проверки состояния Linux-систем

**Тип:** Инструмент для тестирования  
**Язык:** Java  21
**Контекст:** Pet / рабочий инструмент

### Назначение
SSH Inspector — библиотека для выполнения команд на Linux-хостах по SSH
и анализа их вывода в структурированном виде.

Инструмент используется для:
- проверки сетевой конфигурации;
- диагностики состояния интерфейсов;
- валидации параметров системы в автотестах.

### Что реализовано
- Подключение к хостам по SSH.
- Выполнение системных команд (`ip`, `bridge`, `cat`, и др.).
- Парсинг вывода команд в DTO.
- Валидация сетевых параметров (IP-адреса, VLAN, MTU, состояния интерфейсов).
- Обработка неизвестных и нестандартных значений.

### Применение в тестировании
- Использование как backend-проверок в автотестах.
- Проверка состояния системы после UI- и API-действий.
- Диагностика тестовых окружений и CI.

### Инженерные принципы
- Fail-fast валидация данных.
- Строгая типизация DTO.
- Расширяемость парсеров.
- Отделение исполнения команд от логики анализа.
