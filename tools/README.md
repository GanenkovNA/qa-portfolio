# Инструменты для тестирования

В этом разделе собраны вспомогательные инструменты,
которые я разрабатывал для поддержки тестирования и автоматизации.

Инструменты используются для:
- диагностики состояния системы;
- проверки конфигураций и окружений;
- поддержки автоматизированных тестов.

В отличие от автотестов, инструменты не проверяют конкретные сценарии,
а предоставляют инфраструктуру и данные для тестирования.
