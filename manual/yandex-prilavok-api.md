## Яндекс.Прилавок — ручное тестирование API

**Тип:** Ручное тестирование  
**Платформа:** REST API  
**Формат:** Учебный проект

### Объект тестирования
REST API сервисов Яндекс.Прилавок.

### Что было сделано
- Составлены чек-листы для тестирования API.
- Выполнено ручное тестирование API с использованием Postman.
- Проверена корректность обработки параметров запросов и ошибок.
- Проведено негативное тестирование API.
- Подготовлен итоговый отчёт о готовности модулей к релизу.

### Применённые техники тест-дизайна
- Классы эквивалентности
- Анализ граничных значений
- Негативное тестирование
- Комбинаторика параметров запросов

### Результаты
- Выполнено более 130 проверок.
- Найдено более 50 дефектов, включая критические.
- Сделан вывод о неготовности API к релизу из-за некорректной обработки исключений.
