# Ручное тестирование

В этом разделе собраны проекты по ручному тестированию web, mobile и API.
Проекты отражают мой подход к тест-анализу, проектированию тестов и работе с дефектами.

Основной фокус:
- тестирование по требованиям и макетам;
- применение техник тест-дизайна;
- сценарное и исследовательское тестирование;
- оформление воспроизводимых баг-репортов.

---

## Используемые техники тест-дизайна
- Классы эквивалентности
- Анализ граничных значений
- Позитивное и негативное тестирование
- Сценарное тестирование
- Анализ переходов состояний
- Исследовательское тестирование
