## Яндекс.Маршруты — тестирование веб-приложения

**Тип:** Ручное тестирование  
**Платформа:** Web  
**Формат:** Учебный проект

### Объект тестирования
Веб-приложение «Яндекс.Маршруты».  
Функциональность расчёта маршрутов и пользовательского ввода параметров.

### Что было сделано
- Выполнен тест-анализ требований к функциональности расчёта маршрутов.
- Спроектированы проверки пользовательского ввода.
- Составлены тест-кейсы и выполнено тестирование.
- Найденные дефекты оформлены в виде баг-репортов.

### Применённые техники тест-дизайна
- Классы эквивалентности
- Анализ граничных значений
- Позитивное и негативное тестирование
- Сценарное тестирование
- Оптимизация набора тестов для минимально достаточного покрытия

### Результаты
- Проведено функциональное тестирование ключевой фичи.
- Выявлены дефекты пользовательского ввода и обработки данных.
- Сформирован обоснованный набор тестов без избыточного дублирования.
