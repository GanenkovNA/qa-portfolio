## Яндекс.Самокат — ручное тестирование web и mobile

**Тип:** Ручное тестирование  
**Платформа:** Web и Mobile  
**Формат:** Дипломный проект

### Объект тестирования
Веб- и мобильное приложение «Яндекс.Самокат».

### Что было сделано
- Спроектирован чек-лист для экрана «Статус заказа».
- Составлена таблица проверок валидации для экрана «Сделать заказ».
- Проведено тестирование по требованиям и макетам (Figma).
- Выполнено исследовательское тестирование вне тестовой документации.
- Оформлены баг-репорты.

### Применённые техники тест-дизайна
- Классы эквивалентности
- Анализ граничных значений
- Позитивное и негативное тестирование
- Сценарное тестирование
- Исследовательское тестирование

### Результаты
- Выполнено несколько сотен проверок.
- Оформлено более 50 баг-репортов, включая блокирующие и критические дефекты.
