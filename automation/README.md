# Автоматизированное тестирование

В этом разделе собраны проекты по автоматизации тестирования
на разных уровнях: unit, API и UI.

Проекты отражают мой подход к проектированию автотестов,
выбору корректного уровня тестирования и построению пирамиды автоматизации.

Основной фокус:
- проверка бизнес-логики на уровне unit-тестов;
- автоматизация REST API;
- UI-автотесты для ключевых пользовательских сценариев;
- поддерживаемость и читаемость автотестов.

---

## Используемые техники тест-дизайна
- Классы эквивалентности
- Анализ граничных значений
- Позитивные и негативные сценарии
- Сценарное тестирование

---

## Используемые технологии
- Java
- JUnit 4 / JUnit 5
- Selenium WebDriver
- RestAssured
- Maven
- Docker
- Git / GitHub
