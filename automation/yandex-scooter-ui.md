## Яндекс.Самокат — UI-автоматизация

**Тип:** Автоматизированное тестирование  
**Уровень:** UI  
**Инструменты:** Java, Selenium WebDriver, JUnit 4, Maven  
**Формат:** Учебный проект

### Объект тестирования
Веб-интерфейс сервиса «Яндекс.Самокат».

### Что было сделано
- Автоматизированы основные пользовательские сценарии оформления заказа.
- Реализованы проверки навигации через кнопки интерфейса.
- Проверена корректность работы FAQ и логотипов.
- Настроена конфигурация браузера и драйверов.

### Применённые техники тест-дизайна
- Сценарное тестирование
- Позитивные пользовательские сценарии

### Архитектура и подход
- Структурирование тестов по сценариям.
- Выделение конфигурации запуска браузера.
