# QA Portfolio — Никита Ганенков

В этом репозитории собраны мои проекты по тестированию:
ручное тестирование, автоматизация UI и API, unit-тестирование.

Портфолио отражает мой профессиональный рост от ручного тестирования
к построению пирамиды автоматизации и работе с качеством на разных уровнях.

---

## Основные направления

### Ручное тестирование
- Web и mobile приложения
- Анализ требований и макетов (Figma)
- Чек-листы, тест-кейсы, таблицы валидации
- Баг-репорты с приоритизацией
- Исследовательское тестирование

### Автоматизированное тестирование
- Unit-тестирование бизнес-логики
- Автоматизация REST API
- UI-автотесты веб-приложений
- Понимание и применение пирамиды тестирования

---

## Техники тест-дизайна
В проектах применялись:
- Классы эквивалентности
- Анализ граничных значений
- Позитивное и негативное тестирование
- Сценарное тестирование
- Анализ переходов состояний (state-based testing)

## Структура репозитория
